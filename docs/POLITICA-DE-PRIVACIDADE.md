**Última atualização:** Maio de 2024

## 1. Introdução

A **Ruphus Tecnologia** (doravante "**nós**", "**nosso**" ou "**Empresa**") respeita sua privacidade e está comprometida em proteger seus dados pessoais.

Esta Política de Privacidade explica como coletamos, usamos, divulgamos e salvaguardamos suas informações quando você utiliza a plataforma PlanoCerto (doravante "**Plataforma**").

**Leia esta política com atenção.** Se tiver alguma dúvida, entre em contato através dos dados no final desta política.

## 6. Conformidade com LGPD

### 6.1 Seus Direitos

Conforme LGPD, você tem direito a:

**Direito de Acesso:**
- Solicitar cópia de todos seus dados pessoais

**Direito de Correção:**
- Corrigir dados inexatos ou incompletos

**Direito de Exclusão:**
- Deletar seus dados ("direito ao esquecimento")
- Com algumas exceções por obrigação legal

**Direito de Portabilidade:**
- Receber dados em formato estruturado
- Transferir para outro serviço

**Direito de Oposição:**
- Opor-se ao uso de seus dados
- Exceto quando legalmente obrigado

**Direito de Revisão:**
- Contestar decisões automatizadas

### 6.2 Como Exercer Seus Direitos

Para exercer qualquer direito:

1. **Acesse sua Conta:** Vá em **Perfil → Privacidade**
2. **Ou envie email:** privacidade@planocerto.com
3. **Inclua:** Seu email, tipo de solicitação, detalhes
4. **Prazo:** Respondemos em até 15 dias úteis (LGPD: até 30 dias)

**Formulário de Solicitação:**
```
Assunto: [Direito LGPD] - [Tipo: Acesso/Correção/Exclusão/Portabilidade/Oposição]
Corpo:
- Seu email:
- Seu nome:
- Descrição da solicitação:
- Dados específicos (se aplicável):
```

## 7. Retenção de Dados

### 7.1 Quanto Tempo Mantemos

| Tipo de Dado | Duração | Motivo |
|---|---|---|
| Conta ativa | Enquanto usar | Necessário para serviço |
| Após cancelamento | 30 dias | Permitir recuperação |
| Dados deletados | 7 dias | Backup/recover |
| Audit logs | 2 anos | Conformidade legal |
| Logs de erro | 90 dias | Debugging |
| Analytics anonimizados | Indefinidamente | Tendências agregadas |
| Backup off-site | Até 1 ano | Disaster recovery |

### 7.2 Exclusão Permanente

Após 30 dias de cancelamento, seus dados são:
- ✓ Removidos do banco de dados principal
- ✓ Removidos dos backups após 90 dias adicionais
- ✓ Não recuperáveis

**Exceções:**
- ✗ Dados retidos por obrigação legal (impostos, etc)
- ✗ Dados em backups offline (até 1 ano)
- ✗ Dados anonimizados para analytics

## 8. Segurança de Dados

### 8.1 Medidas de Segurança

Implementamos:

**Em Trânsito:**
- ✓ HTTPS/TLS 1.2+ para toda comunicação
- ✓ Criptografia de ponta a ponta para dados sensíveis
- ✓ Validação de certificados SSL

**Em Repouso:**
- ✓ Criptografia AES-256 para dados sensíveis
- ✓ Senhas com hash bcrypt (não reversível)
- ✓ Isolamento de dados por tenant (RLS PostgreSQL)

**Infraestrutura:**
- ✓ Firewall e WAF (Web Application Firewall)
- ✓ Atualização regular de software
- ✓ Monitoramento 24/7 de segurança
- ✓ Pen testing anual
- ✓ Autenticação multifator (2FA) disponível

**Acesso:**
- ✓ Controle de acesso baseado em papéis
- ✓ Senhas fortes obrigatórias
- ✓ Limitação de tentativas de login
- ✓ Auditoria de acessos administrativos

### 8.2 Limitações

**Nenhuma segurança é perfeita:**
- ✗ Nenhum sistema é 100% seguro contra hacks
- ✗ Sua senha é sua responsabilidade
- ✗ Não podemos proteger contra phishing
- ✗ Perda de acesso às credenciais é seu risco

**Você é responsável por:**
- ✓ Manter sua senha confidencial
- ✓ Não compartilhar credenciais
- ✓ Usar conexão segura (não WiFi público)
- ✓ Logout ao usar computadores compartilhados
