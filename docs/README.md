## 🚀 Quick Start

### Para Usuários
1. Leia [Primeiros Passos](MANUAL-USUARIO.md#primeiros-passos)
2. Explore o [Dashboard](MANUAL-USUARIO.md#dashboard)
3. Crie seu [Primeiro Plano](MANUAL-USUARIO.md#planos-de-ação-5w2h)

### Para Desenvolvedores
1. Leia [Documentação Técnica - Primeiros Passos](DOCUMENTACAO-TECNICA.md#primeiros-passos)
2. Setup local com `npm install`
3. Configure `.env.local`
4. Rode `npm run dev`
