## Geral

### O que é PlanoCerto?
PlanoCerto é uma plataforma web para gestão de **planos de ação** estruturados pela metodologia **5W2H**. Permite criar, acompanhar e colaborar em projetos com visibilidade total.

### Como acesso a plataforma?
Acesse https://planocerto.vercel.app e faça login com seu email e senha. Se é primeira vez, clique no link do email de convite.

### Qual navegador usar?
Chrome, Firefox, Edge, Safari (versões recentes). Mobile otimizado para tablets.

### Plataforma tem app mobile?
Não ainda. Use navegador mobile ou Web App (próximo). Dashboard otimizado para desktop.

### Como reportar bug?
- Email: suporte@planocerto.com
- Chat: Clique `?` no canto inferior direito
- GitHub: Abra issue se tiver acesso

### Posso exportar meus dados?
Sim. Vá em **"Planos"** → Selecione plano → **"⬇️ Exportar"** → Excel ou PDF.

## Conta e Autenticação

### Esqueci minha senha, como recupero?
1. Clique em **"Esqueceu senha?"** na tela de login
2. Insira seu email
3. Clique em **"Enviar Link"**
4. Verifique email (cheque spam)
5. Clique no link
6. Defina nova senha

### Quanto tempo leva para o email de reset chegar?
Menos de 1 minuto. Se não receber em 5 min, cheque pasta de spam.

### Posso ter múltiplas contas?
Sim, com emails diferentes. Mas recomendamos usar uma só e pedir múltiplos acessos a empresas.

### Como mudo minha senha?
1. Clique em **"👤 Perfil"**
2. Clique em **"Meu Perfil"**
3. Clique em **"Trocar Senha"**
4. Insira senha atual e nova
5. Clique em **"Salvar"**

### Minha conta foi desativada, como reativo?
Entre em contato com admin: suporte@planocerto.com

### Há limite de tempo para login?
Não há limite padrão, mas admin pode definir janela horária (ex: 8h-18h). Verifique com seu gerente.

## Planos de Ação

### Como crio um plano?
1. Clique em **"Novo Plano"** (topo)
2. Preencha:
   - Título: Nome do plano
   - Unidade: Departamento
   - Diretor/Responsável: Nome
   - Objetivo: O que quer alcançar
3. Clique em **"Criar"**

### Qual é a diferença entre Plano e Item?
- **Plano:** Objetivo macro (ex: "Implementar CRM")
- **Item:** Ação específica dentro do plano (ex: "Pesquisar alternativas", "Treinar equipe")

Um plano pode ter dezenas de itens. Itens podem ter sub-itens.

### Como adiciono itens a um plano?
1. Abra o plano
2. Clique em **"+ Adicionar Item"**
3. Preencha 7 campos 5W2H:
   - **O quê** — Ação específica
   - **Por quê** — Justificativa
   - **Onde** — Local/departamento
   - **Quem** — Responsável
   - **Quando** — Data início/fim
   - **Como** — Método/processo
   - **Quanto** — Custo estimado
4. Clique em **"Salvar"**

### Posso criar sub-itens?
Sim! Clique em **"+"** ao lado de um item para criar sub-item (1.1, 1.2, etc).

### Como edito um item?
1. Clique no item
2. Clique em **"Editar"**
3. Altere campos
4. Clique em **"Salvar"**

### Como mudo status de um item?
1. Abra o item
2. Clique no dropdown de status
3. Escolha: ⚪ Não Iniciada, 🟡 Pendente, 🔴 Atraso, 🔵 Em andamento, 🟢 Concluído
4. Pronto! (atualiza automático)

### Que significam os status?
| Status | Significado |
|--------|-----------|
| ⚪ Não Iniciada | Não começou |
| 🟡 Pendente | Aguardando ação/aprovação |
| 🔴 Em andamento (atraso) | Está atrasado |
| 🔵 Em andamento | Dentro do prazo |
| 🟢 Concluído | Finalizado |

### Posso deletar um plano?
Apenas admins podem deletar. Peça para seu gerente. Melhor é arquivar.

### Como arquivo um plano?
1. Abra o plano
2. Clique em **"..."** (menu)
3. Escolha **"Arquivar"**

Planos arquivados não aparecem na lista, mas podem ser restaurados.

### Posso copiar um plano?
Sim! Clique em **"..."** → **"Duplicar"** → Selecione empresa destino.

### Como compartilho um plano com alguém fora da plataforma?
1. Clique em **"🔗 Compartilhar"**
2. Copie o link
3. Envie por email/chat
4. Pessoa acessa sem login (read-only)

## Visualizações

### Qual a diferença entre Tabela, Kanban e Gantt?
- **Tabela:** Lista com todas as colunas, ótimo para edição rápida
- **Kanban:** Visual por status, perfeito para acompanhamento
- **Gantt:** Linha do tempo, veja sobreposições e prazos

### Como uso o Kanban?
1. Abra um plano
2. Clique em **"Kanban"**
3. Arraste itens entre colunas (status)
4. Clique para editar

### Como uso o Gantt?
1. Abra um plano
2. Clique em **"Gantt"**
3. Veja timeline visual
4. Clique nos botões zoom (Mês, Trimestre, Ano)

## Comentários e Colaboração

### Como comento um item?
1. Abra o item
2. Clique na aba **"Comentários"**
3. Digite seu comentário
4. Clique em **"Enviar"**

### Posso mencionar alguém?
Sim! Digite `@nome` no comentário. Sistema autocompleta.

### Quem vê meus comentários?
Todos com acesso ao plano. Se plano é público, também gente com link.

### Como deleto um comentário?
1. Passe mouse sobre comentário
2. Clique em **"..."**
3. Escolha **"Deletar"**

Apenas o autor pode deletar próprio comentário. Admins deletam qualquer um.

## Anexos

### Que tipos de arquivo posso anexar?
PDF, PNG, JPG, XLSX, DOCX, TXT. Máx 10MB por arquivo.

### Como anexo um arquivo?
1. Abra um item
2. Clique na aba **"Anexos"**
3. Clique em **"Adicionar Anexo"** ou arraste arquivo
4. Selecione tipo (NF, Contrato, Relatório, etc)
5. Clique em **"Enviar"**

### Arquivo não sobe, por quê?
- Arquivo > 10MB? Comprima
- Formato não suportado? Converta para PDF
- Sem espaço? Peça admin para limpar antigos

### Posso deletar anexo?
Sim. Clique em **"..."** no anexo → **"Deletar"**.

### Posso compartilhar um arquivo?
Sim. Clique no arquivo → **"Copiar Link"** → Envie.

## Dashboard

### O que significam os KPIs?
- **Total de Planos:** Quantidade total de planos
- **Em Progresso:** Quantos estão em andamento
- **Concluídos:** Quantos já finalizaram
- **Taxa de Conclusão:** % concluído vs total

### Como interpreto a "Status Row"?
Mostra seus planos em andamento com a saúde de cada um (status + progresso visual).

### Como vejo próximos prazos?
Dashboard mostra automático os 7 próximos dias. Clique em **"Ver Tudo"** para período estendido.

### Posso customizar dashboard?

## Módulo Financeiro

### O que é Conta a Pagar?
Registro de despesa que a empresa deve pagar. Pode ter múltiplas parcelas.

### Como crio uma conta?
1. Clique em **"Financeiro"** → **"Contas a Pagar"**
2. Clique em **"Nova Conta"**
3. Preencha:
   - Descrição (ex: "Compra de notebooks")
   - Documento (NF number)
   - Valor total
   - Fornecedor (selecione)
   - Categoria (tipo de despesa)
4. Clique em **"Salvar"**

### Como adiciono parcelas?
1. Abra a conta
2. Clique em **"+ Adicionar Parcela"**
3. Preencha vencimento e valor
4. Clique em **"Salvar"**

### Como registro um pagamento?
1. Abra a parcela
2. Clique em **"Registrar Pagamento"**
3. Preencha:
   - Data do pagamento
   - Valor pago (pode ser parcial)
   - Forma (PIX, Boleto, Dinheiro, etc)
4. Clique em **"Confirmar"**

### Posso pagar parcialmente?
Sim! Se parcela é R$ 1000 e pago R$ 500, insira 500 no campo valor pago.

### Como excluo uma conta?
Admins podem. Melhor é marcar como **"Cancelada"**.

### O que é Fornecedor?
Pessoa/empresa para quem você paga. Centraliza contatos.

### Como crio fornecedor?
1. Clique em **"Financeiro"** → **"Fornecedores"**
2. Clique em **"Novo Fornecedor"**
3. Preencha:
   - Nome
   - CNPJ
   - Email
   - Telefone
   - Endereço
4. Clique em **"Salvar"**

### O que é Categoria de Despesa?
Classificação da despesa (Pessoal, Operacional, Infraestrutura, etc). Facilita relatórios.

### Como crio categoria?
1. Clique em **"Financeiro"** → **"Categorias"**
2. Clique em **"Nova Categoria"**
3. Digite nome
4. Clique em **"Salvar"**

### Como vejo relatório financeiro?
1. Clique em **"Financeiro"** → Dashboard
2. Vê resumo: total em aberto, atrasado, pago este mês, próximos 7 dias

### Posso linkar conta a um plano?
Sim! Ao criar conta, selecione plano (opcional). Útil para rastreabilidade.

## Administração

**Apenas admins/managers podem acessar.**

### Como acesso admin panel?
1. Clique em **"👤 Perfil"**
2. Se é admin, clique em **"Admin"**

### Como crio novo usuário?
1. Clique em **"Usuários"**
2. Clique em **"Novo Usuário"**
3. Preencha:
   - Email
   - Nome
   - Papel (Viewer, User, Manager, Admin)
4. Clique em **"Convitar"**

Sistema envia email com link de confirmação.

### Qual é a diferença entre papéis?
| Papel | Pode fazer |
|-------|-----------|
| Viewer | Ler planos |
| User | Criar/editar planos, ler financeiro |
| Manager | Tudo acima + gerenciar usuários/financeiro |
| Admin | Acesso total |
| Super Admin | Gerenciar múltiplas empresas |

### Como mudo papel de um usuário?
1. Clique em **"Usuários"**
2. Clique no usuário
3. Clique em **"Editar"**
4. Mude papel
5. Clique em **"Salvar"**

### Como defino restrição de área/unidade?
1. Edit usuário
2. Na seção **"Restrições"**, selecione:
   - Áreas (departamentos)
   - Unidades (filiais)
3. Se vazio = sem restrição
4. Clique em **"Salvar"**

### Como defino horário de login?
1. Edit usuário
2. Na seção **"Horário de Login"**, preencha:
   - Início (ex: 08:00)
   - Fim (ex: 18:00)
3. Clique em **"Salvar"**

### Como deleto um usuário?
1. Clique em **"Usuários"**
2. Clique no usuário
3. Clique em **"..."** → **"Remover"**

Dados do usuário mantidos, mas login desativado.

### Como crio papel customizado?
1. Clique em **"Papéis"**
2. Clique em **"Novo Papel"**
3. Preencha:
   - Nome (slug, sem espaços)
   - Label (nome exibido)
   - Descrição
4. Selecione permissões (checkboxes)
5. Clique em **"Salvar"**

### Como edito uma empresa (tenant)?
1. Clique em **"Empresas"**
2. Clique na empresa
3. Clique em **"Editar"**
4. Altere informações
5. Clique em **"Salvar"**

## Configurações

### Como troco de modo (dark/light)?
1. Clique em **"👤 Perfil"**
2. Clique em **"Meu Perfil"**
3. Na seção **"Aparência"**, escolha:
   - Light (branco)
   - Dark (escuro)
   - Sistema (segue computador)

### Como ajusto tamanho de letra?
1. Clique em **"👤 Perfil"**
2. Clique em **"Meu Perfil"**
3. Na seção **"Tamanho de Fonte"**, escolha: Pequeno, Normal, Grande, Muito Grande

### Como integro com Google Calendar?
1. Clique em **"👤 Perfil"**
2. Clique em **"Integrações"**
3. Clique em **"Conectar Google Calendar"**
4. Autorize acesso (popup Google)
5. Pronto! Prazos viram eventos

### Como removo integração Google?
1. Clique em **"Integrações"**
2. Clique em **"Desconectar"** (ao lado do Google Calendar)
3. Confirme

Eventos já criados não são deletados, apenas novos não são criados.

### Como integro com Microsoft Teams?
1. Clique em **"Integrações"**
2. Selecione **"Teams"**
3. Cola URL do webhook (obtem no Teams Admin)
4. Clique em **"Testar"**
5. Se OK, confirma

### Posso receber notificações por email?
Sim! Você pode:
1. Novo item atribuído a você
2. Prazo vencendo amanhã
3. Comentário em item seu
4. Plano compartilhado/duplicado

Para mudar:
1. Clique em **"Perfil"** → **"Notificações"**
2. Ativa/desativa cada tipo

## Troubleshooting

### Página não carrega, vejo "erro"
Solução:
1. Atualize (Ctrl+R ou Cmd+R)
2. Limpe cookies: Dev Tools → Application → Clear Storage
3. Tente outro navegador
4. Se persiste, contate suporte

### Não consigo editar um item
Causas possíveis:
- Você é "Viewer" (apenas leitura)
- Item foi deletado
- Você não tem acesso ao tenant

Solução: Peça para admin aumentar permissão.

### Upload de arquivo falha
Checklist:
- [ ] Arquivo < 10MB?
- [ ] Formato suportado (PDF, PNG, JPG, XLSX, DOCX)?
- [ ] Internet estável?
- [ ] Supabase Storage está OK?

Se tudo OK mas falha, contate suporte.

### Notificação por email não chega
- Checque pasta SPAM
- Aguarde 1-2 minutos
- Verifique se notificação está ativa em **Perfil**
- Verifique se evento realmente ocorreu

### Integração Google Calendar não funciona
- Você autorizou acesso?
- Token expirou? Reconecte
- Google está fora? Checque status.google.com

Solução: Clique em **"Integrações"** e reconecte.

### Não vejo plano de outro usuário
Causas:
- Plano é de outro tenant (empresa)
- Você não tem permissão de leitura
- Plano foi deletado

Solução: Peça para colega compartilhar link público ou admin aumentar acesso.

### Senha não funciona
- Checque CAPS LOCK
- Copie/coleque senha (não digite)
- Se esqueceu, clique em "Esqueceu senha?"

### Admin panel não aparece
Você é "Viewer" ou "User"? Apenas Manager+ pode acessar admin.

## Segurança

### Como meus dados estão seguros?
- ✅ Criptografia em trânsito (HTTPS)
- ✅ Criptografia em repouso (Supabase)
- ✅ Isolamento por tenant (RLS PostgreSQL)
- ✅ Autenticação JWT
- ✅ Auditoria de ações

### Posso deletar minha conta?
Contate suporte: suporte@planocerto.com

Pedido de deleção é irreversível.

### Quem acessa meus dados?
- Você (sempre)
- Outros no seu tenant (se tiver permissão)
- Admins do seu tenant
- Super admins (apenas se necessário para suporte)

Outros tenants não veem seus dados.
